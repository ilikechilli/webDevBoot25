What is Flexbox?

A one dimensional layout method for laying out items in rows and columns. It allows us to space elements dynamically as the viewport changes. You can set the "display" property of the container to "flex" to initialize flexbox for a container. This will change the behavior of the container.

Flex direction

A flexbox has two axes. By default, the main axis in flexbox is from left to right. But this can be changed using "flex-direction": By default this is set to row but the main axis can be set to:

  • row
  • row-reverse
  • column
  • column-reverse

Even if the elements within the flexbox have a width and height set, flexbox may override this if the flexbox container does not have sufficient space to accomodate those dimensions.

Justify content

The Justify content property will determine how the content is distributed across the main axis of a flexbox container. The value is "flex-start" by default and can also be:

  • flex-start
  • flex-end: Take the content and move it to the end of the main axis
  • center: this will center the content on the main axis
  • space-between: it will distribute the empty space evenly BETWEEN the elements
  • space-around: space is distributed around all elements
  • space-evenly

Justify content property is named strangely because the main axis can be both horizontal or vertical depending on how the flexbox container is set up.

Flex wrap

This property will determine what happens to the content of the container if the container gets smaller. By default a flexbox will shrink the elements within but changing the flex-wrap property to "wrap", it will allow the elements to wrap to a new line/column depending on if the flexbox container direction is set to row or column respectively

Align Items

The align-items property determines how content is distributed along the cross-axis. By default the cross axis is in the vertical direction but can be different depending on how the flex-direction is set. Align-items can be set to:

  • center
  • baseline
  • flex-end
  • flex-start

Align-content and Aling-self

Align content is used to distribute space acros the cross-axis. It controls space beween elements on the cross axis. It only does something if the content is wrapped. If it isn't then it does nothing.

Align-self is not applied to the flexbox container but to the elements themselves allowing positional control of individual elements!

Flex-basis, grow and shrink

flex-basis defines the initial size of an element before additional space is distributed.

Flex grow controls the amount of space an element should take up.

Flex-shrink controls how items shrink if they are larger then the container.

Responsive design and Media Queries

The problem is that there are many different devices with different screen sizes and orientations. The goal is to create a single website that is responsive around the device that is displaying them.

The goal is to create a single website with a single stylesheet that is able to respond to different device sizes and features.
