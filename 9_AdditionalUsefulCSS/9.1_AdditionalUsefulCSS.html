Opacity and Alpha channel

Alpha

Alpha channel is an addition to regular RGB colors and adds Alpha to the color which is transparancy.

Consider the example below where there is a div with a magenta background color. That div is populated by three other divs that have a white background color but with varying settings of transparancy or "Alpha" The top left inner square has an alpha setting is 1 (or 100%) and is not transparant at all, the top right square that has its transparancy set to 0.7 (or 70) and lets some of the background color of the parent div through and the lower left square has it set to 0.3 and lets much more of the magenta through:

Opcity

Opacity controls the opacity of an element but also of all its descendents. So if the element should include text, both the background color of the element and the text will be made (partly) transparant

Position

Position is confusing CSS property at first glance. It is very userfull nonetheless. It sets how an element is positioned in a document. It is the "top", "right", "left" and "bottom" properties which determine the final position of an element.

The five settings for position are:

  • static (default): The Elements is positioned according to the normal flow of the document
  • relative: the element is positioned relative to itself after following the normal flow of the document. This one is a bit tricky but is well demonstrated on MDN
  • Absolute: The element is removed from the normal flow and is then positioned relative to its closest ancestor. Also tricky sometimes!
  • fixed: The element is removed from the normal flow and is positioned relative to the viewport
  • Sticky: The element is removed from the normal flow and is positioned relative to its nearest scrolling ancestor.

It is difficult to demonstrate all the positions with elements because for some it would hinder the view of the overall page so please refer to the MDN reference on "position" for a great demo.

One more important point is that an absolute positioned element can be positioned within its parent container if that container is set to relative positioning first!

There is more to Background

Background image

Rather than use a background color, developers can also use background images using the "background-image" property and a URL to refer to the image that you want to use.

Background size

Contain, cover or repeat a background image. You can also specify how an image should repeat.

Background position

This determines where a background image is aligned.

Background

There is a general shorthand property for background that can set color or image and the alignment and size all in one go.

Google Fonts and how to use them

Google fonts offers a great way to implement a wide variety of fonts in your document

Even if the fonts are not included on the users machine, it can be included as a file or can be references from Google fonts.

Navigate to google fonts and select the font(s) that you want to use. Select "embed" to view the code snippet needed to include the code into your project. The snippet should be included in the head of the page.
