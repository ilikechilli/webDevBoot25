Next steps in HTML and semantics

Inline versus block elements

Inline elements will not require their own line and can be placed 'in line' with other elements on the page

Block elements will populate their own new line.

Whether an element is inline or block is predetermined and each element will be one of those two but this can be changed using CSS

Div elements: The content division element

This is a generic container to hold things and group things together. A div is a block style element.
