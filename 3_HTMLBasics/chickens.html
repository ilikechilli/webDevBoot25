Anchor Tags

Anchor tags kinda build the web. The are used to create hyperlinks to other web pages
They introduce a new concept called tag parameters, these are not added between the opening and closing tags but within the opening tag itself and allow programmers to add attributes to tags
The anchor tag needs an 'href' attribute with a link to work properly, otherwise it won't do anything. Clicking this line will take you to google!

Anchor tags can also be used to navigate to another page within your own project! So you don't need to provide a WWW link but you will need to link to the page in your folder.
