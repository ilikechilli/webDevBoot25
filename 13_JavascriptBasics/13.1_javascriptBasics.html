JavaScript Basics

String Methods

Basic string methods (functions that can be performed on strings) are as follows:

  • .length: this is a property and not a method which is made clear by the lack of parentheses
  • .toUpperCase(): Takes a string and creates a new version of it that is all caps.
  • .toLowerCase()
  • .trim(): trims off any white space at the beginning and / or end of a string

Methods can also be chained like this: "string".trim().toUpperCase()

String methods with arguments

Some string methods allow arguments to be passed to the method between the parentheses.

  • .indexOf("Cat"): This method will tell the user the index where a string occurs first.
  • .slice(beginIndex, [endIndex]): Extracts (slices) a portion of a string. The end index is optional and the beginIndex tells the user where to begin extraction
  • .replace("stringToReplace", "stringToInsert")

Random numbers and the Math Object

Pseudo random numbers can be created using "Math.random()" which will return a number between 0 and 1 which can in turn be multiplied by any range to achieve a random number in that range.

The Math Object is built into JavaScript and is a collection of properties and methods for mathematical constants and functions.

Commonly used methods in the Math object are:

  • Math.floor(float)
  • Math.ceil(float)
  • Math.random()

JavaScript Arrays

Introduction to arrays

Arrays are ordered collections of objects

The basic syntax for creating an arrays is = let myArray = []

Basic built in array methods in Javascript

  • .length (This is a property and not a method)
  • .push("something"): The push method adds the something to the end of the array. It also returns the new length of the array after pushing the something to the array.
  • .pop(): This method removes the last item in the array (and returns this item)
  • .shift("something"): This essentially does the same as pop but at the beginning of the array
  • .unshift(): This does the same as push but adds someting at the beginning of the array.
  • concat(): merges arrays
  • .includes("something"): if an array includes a particular value it will return true; it returns false if the array does not contain the searched item.
  • .indexOf(): This returns the index if an item in an array if the array contains this item. It returns -1 if the item is not in the array.
  • .reverse(): This method reverses the array in place. This means that the original array is destroyed and replaced with the reverse.
  • .slice(begin, end): This returns a copy or a portion of an array defined by a begin and up to but not including the end index. The arguments are optional. Providing no indices will return the whole array.
  • .splice(startingindex, number of items to remove, insert): removes a range of items from an array and returns these items as an array. It also modifies the array permanently. Splice can also be used to insert something into an array if provided with a thrird argument
  • .sort(): This method has limited use because the default use to sort things is by converting everything to a string and then comparing the UTF rating to one another. This means that numbers will not be sorted realiably. To sort numbers it is more useful to create our own sorting functions that will be revisited later.

Javascript Loops

FOR-loops

For loops allow the repetition of code for a set number of times. This number is defined when writing the for loop. Writing the for loop involves settin an initial condition for a temporary variable and a condition where the loop should stop. Finally the variable is changed (often incremented or decremented) until it is changed to match the stopping condition. As long as the stop condition evaluates to true, the code will keep going.
