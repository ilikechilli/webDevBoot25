Bootstrap

Bootstrap is CSS that someone else has written to be incorporated into our own code. It is a CSS framework. It is one of many frameworks and is used extensively but other frameworks may also exist so it is possible that you will not work with bootstrap in the future. The purpose of this section is to get you comfortable with using frameworks, not so much needing to use bootstrap.

What is Bootstrap?

Bootstrap is a CSS framework which is a bunch of code written by others that can be used by developers by inserting the components into an app.

Bootstrap is responsive and offers some very nice pre-styled elements such as buttons, navbars and froms that are very nice to use.

The bootstrap Grid system

The bootstrap grid system provides classes called row and col that can be applied depending on the viewport size. Bootstrap has predefined media-queries that can be used.

Including bootstrap

The bootstrap stylesheet can be added as a link element such as the link element for this document but it requires users to download the stylesheet and add it to the project.

Using a CDN is probably much easier and can be included with a simple link through a URL. The CDN link was added to this page.

It is best to add the bootstrap link before your own stylesheet to allow overwriting any bootstrap styles that you want to change.

Cerctain components such as a collapsable navbar in Bootstrap require some javascipt and can be added to the html file as CDN as well.
