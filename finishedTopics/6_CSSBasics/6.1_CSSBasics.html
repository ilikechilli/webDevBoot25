CSS Basics

CSS stands for Cascading Style Sheets and is used to provide styles to our HTML web pages. CSS and HTML go hand in hand.

In this section we will be going over the absolute basics of CSS which include how to include it in our HTML page and how to start styling our html page.

What is CSS

Remember "The Purple Dinosaur Danced"

  • Purple = Adjectives = CSS
  • Dino = Nouns = HTML
  • Danced = Verbs = Javascript

How to include CSS to your HTML

  • Inline (not recommended because all elements need to be styled separately and it clutters things up)
  • In a style element in the head: Better but then it should be copied in every html page
  • In a CSS document: Best because it can be re-used

The styling for this page is added as an external CSS file and is linked to this HTML file in the head section using a <link> element

To include a css to your HTML file, use the code "<link rel="stylesheet" href="path/to/stylesheet">" inside the <head> of the page.

Common CSS Text Properties

  • text-align: Center, left, right...
  • font-weight: Normal, bold...
  • text-decoration: underline, overline, line-through, text-decoration-color, text-decoration-style
  • line-height: The spacing between lines
  • letter-spacing: The spacing between letters

Other basics

  • font-size: pixels, %, ...
  • font-family: Choose a specific font and complement it with a serif or sans-serif
