Javascript Functions

How to write a function

Functions need to be:

  1. Defined: using the keyword "function" and a name for the function. If the function takes arguments, give the arguments a placeholder name some code that the function needs to execute.
  2. Run / Execute: Call the function by typing the name of the function and adding parentheses (and arguments if applicable).

Adding parameters/passing arguments to functions

Functions without parameters will always perform the same code and output. Adding arguments to a function call gives the function more... functionality

Functions can be defined using multiple parameters. This is as simple as adding those parameters when creating the function but keep in mind that this is order-sensitive.

If parameters are defined when creating the function, you can ignore passing some arguments unless they are required by the function in which case you will get an error.

Intermediate function concepts

Scope

Scope refers to variable visbility. Where is a variable seen in the code? You will see that variables defined inide a function declaration are not available to other parts of the script.

Block scope

Using the let keyword to define a function will make sure that the variable is scoped. It will not be available outside of the "block-scope" where the variable is defined. However, if the variable is defined using the "var" keyword, it will be available outside the block scope. It is generally not recommended to use the "var" keyword and define variables within the scope that they are going to be used. Also keep in mind that a "let" variable can be redefined within a block and will only exist there while the original definition remains valid outside that scope. Weird stuff but it's usefull to know!

Lexical scope

An inner function defined withing another function definition has access to the variables defined in the parent function.

Function expressions

Functions can be defined using the function keyword but can also be defined as an expression. This involves storing a function inside a variable!

Higher Order Function

This comprises two main things:

  1. Accept other functions as arguments
  2. Returns a function
