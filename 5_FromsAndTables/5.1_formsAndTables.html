Forms and Tables

Tables

Html allows the creation of tables. Theses tables should be used to display tabular data and not to structure a web page since there are much better tools for that.

Tables exist within a <table> element and can take several children elements, some of which are required for the table creation

  • TR = A row of data
  • TD = A single cell of data
  • T
  • thead
  • tfoot

The following is an example of a table listing large birds

Type of bird Average weight kg (lbs) Max weight Flighted
Ostrich 104kg (230 lbs) 156.4 kg no
Somali Ostrich 90 kg (200 lbs) 130 kg no
Wild turkey 13.5 39 yes

Colspan and rowspan

The table above is semantically correct but not yet styled. It is also possible to have several columns or sows span differently using the <colspan> and <rowspan> elements.
